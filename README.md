# Cairn — Documentation

Docs site for [Cairn](https://github.com/Enoch208/cairn) — proof-carrying memory for AI agents, built on Sui · Walrus · Seal · MemWal.

Built with [Mintlify](https://mintlify.com).

## Local preview

```bash
npm i -g mint
mint dev
```

Open http://localhost:3000. Edit `docs.json` for navigation/branding and the `.mdx` files for content.
